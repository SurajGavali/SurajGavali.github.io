Installing XV6 Operating system



Here are the steps:

Step-1 : Installing qemu

$ sudo apt install qemu



Step-2 : Installing xv6

$ git clone git://github.com/mit-pdos/xv6-public.git



Step-3 : Compiling xv6

$ make



Step-4 : Compile and run the emulator qemu

$ make qemu


while running above command if you get an following error!

Error: Couldn't find a working QEMU executable.

Is the directory containing the qemu binary in your PATH or have you tried setting the QEMU variable in Makefile?



Simply run following command :

sudo apt-get install ipxe-qemu ipxe-qemu-256k-compat-efi-roms qemu-block-extra:amd64 qemu-kvm qemu-system-common qemu-system-data qemu-system-gui:amd64 qemu-system-x86 qemu-utils


The emulator will start but it will echo on the terminal too. Better to run it with the following frag (preferred):

$ make qemu-nox
